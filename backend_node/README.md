# Chat Backend

Backend simples para aplicação de chat usando Express e TypeScript.

## Instalação

```bash
npm install
```

## Executar

```bash
npm run dev
```

O servidor irá rodar em `http://localhost:3001`

## Endpoints

-   `POST /api/chat` - Enviar mensagem para o bot
-   `GET /health` - Verificar status do servidor
